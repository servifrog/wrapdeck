# Publication Notes

This repository is a curated public source release for WrapDeck.

The public source is intentionally limited to:

- first-party static runtime files
- public privacy, security, deployment, and product documentation
- local verification scripts
- license, notice, and brand-use notes

Generated deploy output, private operations notes, source-pack material, local
tooling residue, account material, and machine-specific paths are intentionally
excluded.
