WrapDeck 60-second guide

How To Use WrapDeck In 60 Seconds

Fake examples only. Keep official records in the usual workflow.

Short version: Pop WrapDeck into its own skinny side window, load a saved board or templates, then pick a Full Template or build the note piece by piece.

1. Park WrapDeck beside your work

Open WrapDeck in its own window, make it as skinny as practical, and keep it beside the page or system you are working in. The main app stays usable down to a narrow sidecar width.

2. Load a board or templates

Paste a setup string to restore a saved board, or paste one or more full templates. Press Load Setup or Templates. WrapDeck creates Full Template buttons and optional reusable phrase buttons for this browser tab.

A setup string reloads buttons and layout. Template lines create new Full Template buttons. Neither one reloads a customer's call note.

CALL START// verified greeting// explained options// CALL END

3. Choose the fastest note tool

Use a Full Template when the whole note already fits the call. Use Build As You Go when you need to add details one at a time. Use Quick Forms for separate copyable form text that should not change the call note.

4. Build as you go

Type a note piece and press Enter for something specific to this call. Use the pencil, arrows, and x controls to edit, move, or remove note pieces. Open reusable phrases only when you want to browse saved phrase buttons. Use Save Full Template in the dock when a built note should become a reusable Full Template. Use the Scratchpad tab on Build As You Go when you need temporary call notes beside the detail box.

5. Know the small controls

  • Reusable phrase x hides that reusable button for this session.
  • Current note x removes that phrase from the current note.
  • Current note arrows move a phrase up or down in the current note.
  • Current note pencil edits one piece without changing templates.
  • Search filters only narrow what you see. They do not save, delete, or change your buttons.

6. Finish and copy

Choose Call Ended or Transferred To. For transfers, enter a destination before copying. The bottom dock is where you preview, edit, start a new call, and copy the valid note.

Keyboard Flow

  • Enter in Add note piece adds that one-call phrase.
  • Shift+Enter in Add note piece adds a newline while typing.
  • Enter in a current-piece edit saves that piece; Escape cancels it.
  • Enter in a valid transfer destination moves focus to Copy.
  • Escape exits temporary final-note edit.

Session-Only Reminder

WrapDeck runs in this tab. It does not save or upload what you type. Refreshing or closing the tab clears the session. Scratchpad text is temporary and is not copied or restored by setup strings. Saved non-call reference text and Full Templates you explicitly save from built notes can be restored by setup strings.
